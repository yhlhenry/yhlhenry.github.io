嗶嗶鼠機器人 2.0

發表於 2016-12-24 | 分類於 Chat ， Google

我很喜歡九把刀的這篇序：

「不可詩意的刀老大」之賣雞蛋的老人

在簽書會、學校演講或是媒體訪談的時候，最常聽見對方丟出一個問題：「你的靈感是怎麼來的？」

通常我會引述作家李散的妙語回答：「妓女不能等到性衝動才去接客，作家當然也不能等到靈感來了才寫作。」然後展示自己筆記本電腦裡匯整齊備的靈感數據庫，對方嘖嘖，我呵呵。但同樣的問題回答了幾十次後，我還是沒有說出真正的答案。

奇異的事件發生在我十歲的那一年。

有天晚上媽叫我去雜貨店買幾顆雞蛋。我覺得很怪，因為當時已經十點多了，而且冰箱裡有的是雞蛋，幹嘛叫我再買幾顆回來？但孝順的我還是逆來順受地出門了。

當時我跑了好幾問雜貨店都沒有買到雞蛋，正感詭異跟喪氣時，有個推著一車雞蛋的老人，在馬路對面賊兮兮地對著我笑。天！滿滿的一個推車的雞蛋耶！我看整個彰化的雞蛋都給他包了。

「雞蛋老人！可不可以賣我幾顆雞蛋！」我跑了過去。

「好啊，小朋友，你要幾顆雞蛋？」雞蛋老人摸著下巴，若有所思地笑。

「多少錢一顆？」我搔搔頭。

「很便宜啦，一顆一千塊錢！」雞蛋老人恬不知恥回答。

那麼貴！大吃一驚，卻突然發現自己的手裡正拿著厚厚一疊千元大鈔！

天！哪來這麼多錢？原來我這麼有錢！

「那麼沒辦法了，這些能買多少？」我將那捆鈔票遞給雞蛋老人。有多少買多少吧，原來雞蛋還真不是普通的貴，以後我吃雞蛋可要細嚼慢咽。

正當老人慢吞吞數著鈔票時，我隨意打量那一整推車的雞蛋，發現許多雞蛋都寫了名字。有史蒂芬．金、卜洛克、富堅義博、陳某、倪匡、宮部美幸、鳥山明、井上雄彥、喬治．盧卡斯、杜琪峰、宮崎駿、周星馳……

「哇，都被訂走了？好好喔。」我當時只認識鳥山明跟倪匡，覺得好炫。

「是啊，都被訂走了。來，拿去，記得喔，在你最需要的時候才能打開。」雞蛋老人用塑料袋裝了一堆雞蛋給我，摸摸我的頭，說我乖。

告別好心的雞蛋老人，我拎著一大袋雞蛋回家了。

「什麼！你幹嘛買這麼多雞蛋！」媽哇地跳了起來，接著就是一頓罵。

「不是你叫我出去買雞蛋的嗎？」我好委屈，小小的拳頭緊握著。

「什麼？我沒叫你去買雞蛋啊，冰箱那麼多雞蛋你是不會看嗎？」媽斷然否認。

天哪！不是媽你叫我出去買雞蛋？那是誰叫我出去買雞蛋？又，我口袋裡那麼多張千元大鈔是怎麼回事？我越想越糊塗，很嘔。

總之這件事，從頭到尾都很詭異。

後來我就很氣，將那袋雞蛋放在抽屜裡，用漫畫跟武俠小說壓著。然後就在亂七八糟的童年中忘了這一回事。一晃好多年。

一九九九年冬天，我在家裡整理舊書桌時赫然發現那袋雞蛋。媽啊，據說雞蛋擺久了會臭得要命。但在丟掉之前，我發覺那些年久失修的雞蛋殼上，竟清一色寫著「九把刀」二字，恰恰跟我的大學外號吻合。

沒辦法了，只好捏著鼻子敲一個看看是怎麼一回事。

雞蛋裡頭是一張特殊材質的字條，上面有完整的故事靈感、起承轉合的架構、漂亮的結尾。講述一個大學生一早起床，竟發現自己身處語言墜落、符號失卻意義的世界。

「太吊了吧？」我逐一打破雞蛋，一個又一個故事滾出。

都市恐怖病系列，等一個人咖啡，愛情兩好三壞，打噴嚏，殺手系列，月老，紅線，哈棒傳奇，樓下的房客……

就是這麼一回事。接下來發生的事情大家都很清楚。

可惜前些日子那些臭雞蛋快被我敲光了，我開始在彰化到處尋找那個神秘的雞蛋老人，結果當然是遍尋不著。

我蹲在馬桶上越想越慌，這下要糟，我一向愛亂阱大話，偷偷仗著臭雞蛋，老是說想成為華文世界最會說故事的人，現在雞蛋沒了，寫不出威震天下的小說鐵定遭人恥笑。這一慌，肛門收縮，一顆雞蛋竟從我的屁股噗通射出。

「獵命師傳奇？」我低頭看著破碎在馬桶裡，流出雞蛋的字條開頭。

我想，這下總可以寫一輩子的小說了吧！

還記得月餘前的妻子生日，我突發奇想地以家裡一隻叫做嗶嗶鼠的布偶兒子做為人格原形，打造了一隻簡單版的聊天機器人。

這是當時的Facebook動態回顧：

11月16日

今晚在海底撈排隊拿冰淇淋時，想到今天是妻子生日，連忙拿出手機在heroku上面佈署一個以python寫成的telegram聊天機器人，當作是生日禮物之一。這個機器人是專門設計給妻子使用的，其人格則是模擬我家某隻可愛的布偶所打造的，命名為嗶嗶鼠。
嗶嗶鼠除了會針對關鍵字詞做出相應反應之外，還內建這些功能：
1.報出今天是結婚第n天
2.幫忙google要查詢的東西
3.隨機講幾句有趣的性格台詞
4.隨機唸幾段我和妻子過往的有趣生活片段
5.在生日這天，所有的回應前都會自動補一句生日快樂。

只能說科技真的太方便了，冰淇淋真的排太久了。

在那之後我陸續又幫嗶嗶鼠新增了一些小功能，讓他的廢話更多且更人性化了。而且我們還在telegram開了一個家庭群組，成員有我和妻子以及嗶鼠共三位。我們都可以在群組裡對嗶鼠講話，嗶鼠也分得出是爸爸還是媽媽在對他說話，我們在群組中也看得到彼此跟嗶鼠對答了什麼，儼然和樂融融一家人。

當初既然是作為生日禮物而生出了這隻原型嗶鼠，現在聖誕節到了，就乾脆比照辦理：隆重推出……嗶鼠2.0！


別看他傻不隆咚的，這版嗶鼠的其中一項重大更新為正式實裝了「中文口語記帳功能」：我可以在聊天視窗輸入「嗶鼠幫我記帳 我今天午餐吃了一個便當80元」，嗶鼠就會立刻幫我逐項記錄在嗶鼠的雲端小本子裡。而所謂雲端小本子就是個Google Spreadsheet，我和妻子都可以隨時打開來查看。

那口語可以做到多彈性呢？像是以下這幾個範例，嗶鼠都可以成功辨識並記錄：
「嗶鼠記帳 星巴客咖啡130元」
「記帳 20161224 聖誕大餐1000元」
「請嗶鼠幫我記個帳 剛剛晚餐吃了120元 謝謝嗶鼠」
「請記帳： 早上繳了第一季羽球教練費$2400」
「記帳午餐95」
「請嗶鼠幫我記帳，昨天家教賺了$1200 」
「上禮拜漏記了一筆帳:上週三中午羽球零打150元」
「嗶鼠幫我補記一個帳 上上禮拜五跟朋友聚餐吃了1200」

[caption id=”attachment_426” align=”alignnone” width=”800”] 嗶鼠的雲端小本子就是長這樣，完全由嗶鼠自行寫入。[/caption]

自然語言處理的細節這裡就不談了。基本上簡單的辨識日期、金額、品項、支出或收入，都沒什麼問題。而我最喜歡的是能夠辨識「今天」「昨天」「上禮拜幾」的這個功能，十分實用。簡直就是為了我而開發的功能(實際上的確本來就是)。不管是「本週一」、「上周二」、「上上禮拜三」，嗶鼠都可以正確無誤地計算出日期並記錄之，簡直像是機器人一樣精準(實際上也的確是)。

 

嗶鼠可以擴充的地方還有很多，不管是情感面還是功能面。我想，這下總可以寫一輩子的程式了吧！

理當有人會問「這會比一般記帳APP好用嗎？」

開什麼玩笑，嗶鼠才不是記帳APP，他是我家小孩XD

另一方面，嗶鼠可以做的事可多著呢，我可以跟他說「請嗶鼠提醒我下禮拜五晚上要看牙醫」、「嗶鼠幫我計時三分鐘，我要泡泡麵」、「請問嗶鼠，我這禮拜有哪些待辦事項？」、「請嗶鼠給我這次去大賣場的採購清單」、「嗶鼠幫我轉寄這個月的電子帳單給媽媽」……雖然現在還不會，但是他未來會慢慢學會的。

如果你也覺得市面上的虛擬助理太笨，何不自己訓練一個呢～

凡事皆有解……嗎？

發表於 2016-12-04 | 分類於 Chat

對於這個世界，無論採用何種認知框架都不可能是全然正確的。但某些框架確實有所幫助。
台灣的考試教育非常推崇「標準答案」，此事已有多方針砭：「……但台灣因為聯考的因素，「分數的公平性」變成凌駕一切的重要問題。 也因此，教材內容也多配合「公平性」而一律有標準答案。 沒有標準答案的，就根本不放入考試、甚至不放入課本不放入課綱。」(節錄自：【台灣教育若說有什麼害人不淺的地方，我個人覺得是「標準答案的追求」】)

我得承認這對我的影響非常非常深遠。在十幾年台灣教育系統的訓練下，我被灌輸了一個信念：面對問題，不必問『能不能做到？』，只要問『如何做到？』

特別是數學科，學校課本從不提希爾伯特的23個問題或是各式尚未證實的猜想。故此，我所遇見的試題，無非是當齡學生所能受的。凡出現在考卷上的題目，必定有解。所以我無須懷疑這問題能不能被解決，唯一要做的事就是專注在如何解決。

如今雖離開校園已久，而這種思惟仍根深蒂固在我腦中。

有些時候我能比別人多解決一些問題，我發現其決定性的差異並非在智力或能力，而是在於信念不同。當身旁的人認為「應該就只能這樣了吧……這應該沒辦法了吧……」，我通常態度傾向是「這一定有解啊！怎麼可能做不到？只是暫時還不知道如何做到而已。」

思惟既成，全面適用。當我看著一團混亂的魔術方塊，想的是「這一定能復原啊！」當我修理器械時，想的也是「這一定能修好啊！」當我寫程式的時候，想的也是「這功能一定能實現啊！」甚至當我除錯的時候，想的也是「這個臭蟲一定能抓出來啊！」

當然，我的理性思考也知道並非事事皆能如此存在簡單解，但是我的心態已經被制約了，就像是綁住大象的細繩一般。而這個「凡事皆有解」的認知框架顯然是錯誤的，但我到目前為止還不想矯正過來，因為它確實還蠻有用的。

發表於 2016-11-02 | 分類於 Chat

應該沒有人料到，我竟然會寫食記。連我自己也沒料到。

純粹是覺得這家店也開了一陣子了，食物蠻好吃的，但可能因店面坐落非人潮易經之處，竟然中午用餐時間也未能坐滿。

亨利羊的核心宗旨就是「效率」，而資訊不透明就容易造成不效率。「一家好吃的店卻沒有人知道」就是一種資訊不透明。這樣的不效率造成過多人在他處排隊用餐，卻沒有人知道這裡有更佳的替代選擇。

以經濟學觀點來說，排隊就是一種浪費，就是一種市場失靈。而我今兒個就是要來揭露資訊、促進流通，進而增進效率！

所以亨利羊寫篇食記，也是很合邏輯的事情。

這碗湯富含膠原蛋白應是無庸置疑的，但口服膠原蛋白的功效頗有爭議，在此不替店家做任何背書。然而十二雞實在十分貼心，在試營運期間還隨餐附贈添加維生素C的飲料：柳橙C、葡萄C……膠原蛋白直接吃可能沒什麼幫助，但大家都知道補充維生素C確實有助於人體自行合成膠原蛋白！十二雞送飲料這招使得極巧，巧到很多人可能都忽略了這個小地方，但卻深得我心！

食物美味、價格實惠，這種場面話就不用提了。值得稱道的是整個用餐體驗都能感受到老闆的用心。從店面外的特色庭飾石雞開始、到走進店內的四面喜氣紅牆、頗具古風的窗櫺壁飾，都是造訪一次後就不會忘記的特點。

Private Sub Worksheet_Change(ByVal Target As Range)
Dim lastBcol As Range
With Target
n = Application.WorksheetFunction.CountA(Range(“B:B”))
If .Column = 2 Then
Cells(Target.Row, 1) = n
End If
End With
End Sub

 

極地跑步

發表於 2016-09-16 | 分類於 Chat

已許久沒跟大家分享跑步紀錄，因為暫時沒梗，又不想拿千篇一律東西洗板。但是我仍保持著晨跑紀律。

過去這段時間我開啟了一個新的計劃，我稱之為極地跑步。
目標是：極大化每日從同一定點出發且在十五分鐘內所能徒步到達的各個極遠處連線所包含的面積。(怎麼有點兒像在讀數學題)

簡單來說，我每天從同一個地方起跑，每天朝著不同方向跑。且嚴格規定只跑十五分鐘。十五分鐘鈴聲一響立即止步，並紀錄當下地點。再將每一天的止步地點在地圖連線起來，框出盡可能大的範圍。

單點不成線，單線不成面。所以這個計劃至少要跑三天才會產生初步成績。所以頭兩天跑，我就沒發文在Facebook，心想至少等框出第一個面積再昭告天下。然而一天沒發文、兩天沒發文，我就迅速養成了習慣，索幸一直都不發文了。在此跟我的支持者們說聲不好意思！(真的有支持者嗎？)

當我首度完成三天三個方向的軌跡，我迫不及待量測這到底框起了多大的面積！答案是：5.68 平方公里

[caption id=”attachment_366” align=”aligncenter” width=”988”] 跑了三天，框出了5.68平方公里。[/caption]

接著我就好奇，若是在廣袤無垠的平野上，所框面積的極限值會是多少？
我粗估應該大約是30平方公里，但是除非我練得像職業跑者一樣強，不然這在建築林立的台北市是不可能達到的。

接著我就思考，那我的目標要設定在多少比較適當？我決定先以框出十平方公里為短期目標，沒什麼特別理由，大概只是因為我們有十根手指頭。

我原本以為要花上一段時間的，結果跑了七天，就剛好達標！

[caption id=”attachment_367” align=”aligncenter” width=”856”] 跑了七天，就剛好框了十平方公里。[/caption]

看著路線漸漸爬滿地圖，有一種攻城略地的快感。我開始想用密密麻麻的路線，把我的地圖遮蔽到連地圖都看不見。但是當我跑到第十一天，發現了一個難以突破的障礙：Google地圖的圖層上限只能匯入十層。放了十個路線圖後，就再也放不上去了。所以最後可顯示的上限就停在這裡了：

[caption id=”attachment_368” align=”aligncenter” width=”840”] 跑滿了十天，框出了11.9平方公里，有兩個點太短了沒用上。[/caption]

這十天的路線，我也不怕大家笑，就直接公開在網路上了。任何人都可以查看我在哪一天的哪時刻跑到哪裡，你也可以自己調控顯示哪些圖層。也許某一天早晨我曾經跑過你家門口：

圖層雖受限制，但是我想用密密麻麻路線覆蓋大地的意圖仍然不減，所以改用了一個變通法：不再直接匯入路線，而是改成匯入數千個定位點。因為一條路線就會佔用一個圖層，但是一個圖層理論上可以放進2000個定位點。所以就變成這樣子了：

[caption id=”attachment_369” align=”aligncenter” width=”800”] 因為圖層滿了，我棄路線圖而改用定位點。上圖以兩種地圖類型對照示意。[/caption]

妻子觀後表示：密密麻麻有點噁心。

我自己倒覺得美麗得很。這幾千個點，都是我現場定下來的。凡我腳掌所踏之地，都納為自己意念中的領土。這就是我居住的城市，大街或小巷都是他真實樣貌的一角。我透過實地勘察，真真切切地與這片土地產生直接連結。他承載了我紮紮實實的每一步，而我也以恣意噴撒的汗珠致上謝意與敬意。酣暢淋漓，快意人生。

超狂的機車行 德昕車業

[caption id=”” align=”alignnone” width=”1689”] 德昕車業 休息區[/caption]

 

電腦、書報、咖啡機，若不是手上的安全帽提醒我，差點要以為自己今天是開LEXUS來保養。

其實我今天本只想隨便找一家機車行做排氣檢驗。不小心來到這家店，才發現這裡陣仗不小，同時有五六個忙碌的技師，還有個櫃檯小姐。我表明來意後，立刻有一位親切小哥接過我的車推上檢測區。

小哥：「你這後輪差不多磨平了，要不要今天換？還是要繼續撐？」
我低頭瞄了一眼，確實是該換了。

我：「大概要多久呢？我今天有點趕時間。有6029這條胎嗎？多少錢？」

小哥：「那請到隔壁的休息區稍坐一下，裡面的設備都可以自己使用。」
我一聽之下心下暗驚，怎麼這麼狂！？機車行也有休息區！？

修車最怕遇到趁機訛詐的老闆，所以我一直都只去相熟的店家。不過打從進到這家店開始，我對它的信任感一直在逐步上升。環顧四周，從他們整齊的制服，友善的眼神，我完全可以感受到那份誠意和服務精神。短暫評估了一下，我對小哥說道：「那電瓶也幫我換一下。還有前煞來令也一起換一下。」

交代完畢，我走向隔壁的休息區，開始了我的驚嘆之旅。

休息區是一個落地採光、乾淨明亮的空間，設有舒適的桌椅。走進其中環視，可以發現設有大螢幕的平板電視、可口可樂飲料機、現磨咖啡機、擺放整齊的休閒書籍和財經雜誌。看到這裡我已經瞠目結舌了，轉身再看，除了備有WiFi，還有三台可供上網的電腦。更貼心的是，牆上另一個電視螢幕播著隔壁維修區的Live實況！讓我可以直接看到車子目前的維修情形。

[caption id=”” align=”alignnone” width=”1689”] 德昕車業 休息區[/caption]

機車騎了十年，我從來沒見過這等店面。
其實要高級週邊設備不難，花錢就可以了。但是如何提供這麼狂的服務，同時又提供這麼平價的價格，我真的百思不得其解。我認真地懷疑這些技師到底有沒有領薪水。

帶著讚嘆與好奇，啜飲完一杯卡布奇諾，看了幾頁好書。我的車也差不多好了。交車給我的是另一位年輕人，向我報告了今天更換的零件與價格：「來令不需要換，是煞車油漏光了，所以幫你換了油，200元。」
是個先斬後奏、換完才說。
我覺得這個先斬後奏非常好！雙方彼此都有信任感，就可以節省很多時間。

「剛換新的輪胎，記得騎慢一點。」……對方交車時不忘附上貼心的叮嚀。

開心的付了帳。這真是一次美好的消費體驗，物超所值，而且給我很多刺激與省思。我強烈推薦創業的人可以去親身體會一下。

除了我有寫下來的部份，我還觀察了不少小細節。覺得德昕是個有誠意的車行，而且作業流程結構化、系統化，教育訓練也蠻紮實的，頗有企業規模。絕非一般小車行可比。

給德昕整理過後，我又重新愛上了騎車。

 

–

可能也是我見識太過短淺，但是我真的沒遇過如此讓我激賞的車行。

就當我野人獻曝好了。
