亨利羊的1000000種生活效率偏方_0000001_搶時間按電梯

發表於 2015-11-26 | 分類於 Interesting Efficient Viewpoint ， Knowledge

每次等電梯都要等很久，如果可以愈早按下呼叫電梯的按鈕，當然可以愈早搭到電梯。那要如何盡快搶按到電梯呢？除了加快步伐之外，有沒有其他方法呢？
在此提供一個優雅小方法：在朝著電梯按鈕方向等速行進的前提下，算好時機向前抬起手，至與電梯按鈕等高處、延伸手臂同時張開手掌，此時中指自然成為人體最前緣，與此同時剛好走到按鈕前，指尖自然而然撞上按鈕，完成呼叫電梯的動作。切記！要使用中指，絕對比食指更快碰到按鈕！

以此達成在不改變步速的情況下，最短化按到電梯的時間。減少錯過班次的機率。

那「因此趕上原本會錯過的電梯」的機率有多少呢？為方便計算，我們姑且進行以下假設：
1.等待電梯時間的機率分配為指數分配
2.電梯平均每2分鐘來一班
3.使用這種刻意延伸肢體的方式，可以提早0.1秒按到電梯

在此我不做冗長的計算，直接講白話結論：原本電梯在0.1秒內來的機率為1-e^[-(1/600)/2]=1-0.99916701379=0.00083298621
也就是說若我成功提早了0.1秒按到電梯，我因此少等一班電梯的機率是0.08%左右。
因此節省的時間期望值為0.08% x 120 sec = 0.0999583452 sec
假設我每天要按4次電梯，一個月要按20天，一年下來可節省的時間期望值為4x20x12x0.0999583452=95.960011392 sec

數學就算到這裡，意涵請各位自行延伸。

【Alphabear 找單字小工具】(非攻略、非破解)

警告：使用本工具可能導致遊戲樂趣喪失！

我從來沒寫過遊戲評測文，介紹遊戲也不是本站主要走向。但是我覺得透過這個alphabear找單字小工具，得以同時體現「效率」與「創意」兩大核心價值，故為文以誌之。

Alphabear是一款手機上的拼字小遊戲，除了單純拼英文單字之外，它還融合了幾個會讓你上癮的元素，例如練功升級、蒐集熊種、講究實力又略帶運氣的玩法……老實說我覺得遊戲平衡度設計的還不錯，就算卡關了，稍微把熊的等級練高就很有機會過關。不過偶有幾次遇到「只差一點點就過關了，但是有些單字就是怎麼也拼不出來」的窘境，於是我就用excel寫了這個找單字小工具。

下載點： http://bit.ly/1NMMyVY

【功能】
告訴Excel你要找包含哪些字母的單字，程式就會自動列出所有符合條件的單字。例如你場面上可以用的字母有”z”有”a”有”o”，就在輸入欄位打上”zao”，按下確定，即可從清單中挑一個喜歡的單字來用。
【說明】
1.A欄為單字資料庫，目前我隨便放了一個好像是TOEFL還是TOEIC的字庫。使用者可自行擴充。

2.輸入時可不考慮順序。”abc”和”cba”都可以找到所有含有這三個字母的單字。

3.可輸入重複的字母。輸入”a”，會找到所有含有字母a的單字。輸入”aa”則會找到所有含有兩個a以上的單字。輸入”aaa”則會找到所有含有三個a以上的單字。以此類推。

4.必須允許巨集。若您有疑慮，可自行將下列程式碼複製回去修改。

5.受限於電腦運算速度，以及受限於我粗糙的演算法，執行效率可能不盡理想。故不推薦拿來玩90秒的限時關卡。

就業服務乙級技術士考取心得

發表於 2015-11-18 | 分類於 Chat

【就業服務乙級技術士_考取心得】
【前言】我本身高中讀自然組、大學唸社會科學院。除了學過一點統計學，就業服務的考試內容基本上對我來說完全都是全新的科目，一切從零開始讀。所以不管你是來自什麼背景、從事什麼行業，都不必擔心、也別找藉口，靠業餘自學考上的人所在多有，就看有沒有心而已。

這個考試很簡單，分學科和術科，兩科分數同時達60分以上就算是通過考試。為什麼說這個考試很簡單？因為
1.不必和別人競爭。
在公務員考試或研究所考試，你只要差個一分，名次就掉了十幾名，可能就從正取變備取甚至變落榜。但是考就服乙級不必擔心這個問題，考場內都是你的同伴而非敵人，就算所有的考生都比你優秀，你只要想辦法讓自己高於60分就好。簡單吧！

2.不必考100分、也不必考八九十分。你只要考60分就好。
有稍微瞄過考試內容的都知道，課本超級厚、講義超級多，短時間根本不可能背好背滿。還好，課本各章節也是有分「超重要」「普通重要」「不太重要」「看看就好」不同等級，不必全部死背起來，只要從最重要的開始依序記憶，背個七成八成就差不多了。考滿分證照也不會鑲金邊，我們的目標是考過就好。
【考照必勝秘訣】

1.決心
你是認真下決心一定要考到？還是你只是抱持著試試看的心態？如果是後者，我建議還是不要浪費時間金錢在這上面了，去買樂透也許比較划算。

2.紀律

你可能覺得這些秘訣很爛，但我還真的想不出其它有什麼捷徑可以快速考取證照。人生之中想真正把一件事情做好，大概都不脫這幾個要項。

【使用教材】
就業服務乙級技術士技能檢定完全指導手冊(附學術科歷屆試題)
作者：張秋蘭 出版社：全華
(如果你想買的話，可以找我幫訂，我剛好有用不完的優惠券XD)

這本課本看起來每年都會出新版，畢竟法規也是年年更新。如果你把法條都背得滾瓜爛熟，結果發現不小心背到過時的版本，真的會嘔死。所以我建議買最新版的。

雖然我自己用的並不是最新版。

我是2015年7月考上的，但一直都是用2014版的教科書(也就是張秋蘭這本的第三版)因為我不想多花錢買新書，配套措施就是除了課本之外，還要自己隨時準備著最新法條。

推薦使用APP：六法全書
(參考： https://play.google.com/store/apps/details?id=com.mywoo.law&hl=zh_TW )

手機版的六法全書好帶好查，尤其是有時候做考古題，你隱約記得某個東西你有唸過，但就是想不起來是在哪個章節哪個法規。不管翻課本還是翻紙本法條，一時間都不知該從何翻起。此時APP的關鍵字查詢就很好用了！等公車時也不妨拿出來閱讀一下(就暫時別再滑臉書了，等考上再看也不遲，朋友不會不見的)，法規不必強記，就當看故事書一樣，把整個法規從開頭讀到結尾，感興趣的部份就慢慢讀，同時在腦海中想像一下劇情畫面。無聊的部份就快速掃過。你會發現，相較於課本上零碎分佈的單條法令，其實整個從頭到尾看下來之後會更有整體概念。

Excel VBA_依使用者名稱自動切換計算選項(自動計算/手動計算)

發表於 2015-11-12 | 分類於 Excel ， Knowledge

【背景】一來使用函數太多，二來電腦設備老舊，excel計算公式時總是會卡很久。所以我都把excel設定為「手動計算」，等該改的都改好了，再按下F9一次完成計算。

【現象】其它需要共同excel檔案的同事發現怎麼改了儲存格數字，總和卻沒有自動更新。感到不解及困擾。經我解釋原委，並教學如何自行切換為自動計算後，同事仍然感到不解及困擾。

【問題】如何讓excel檔案被同事開啟時是自動計算，但是當我本人開啟時就自動切換為手動計算？

【解法】在ThisWorkbook中寫入以下程式碼

Private Sub Workbook_Open()
 If Application.UserName = "henry.yang" Then
 Application.Calculation = xlCalculationManual
 'MsgBox "自己使用，已自動設定為「手動計算」模式"
 Else
 Application.Calculation = xlCalculationAutomatic
 MsgBox Application.UserName & "您好，已為您切換為「自動計算」模式。"
 End If
 End Sub

認知失調

發表於 2015-11-11 | 分類於 Chat

【認知失調】

認知失調(cognitive dissonance)是個心理學名詞，意指人類的思想和行為不一致時，就會陷入失調因而感到不悅。例如在社會壓力下，我就經常被迫做出違心之舉，以致認知失調。

1.起初我在電話裡報email的時候，”@”這個符號我總是照著唸出來。遇到的不外乎兩種反應：「蛤？」「你是說小老鼠嗎？」，如此反覆好幾次後，我終於屈伏了，要報email的時候直接投降說「小老鼠」。如此省了不少溝通的力氣，但是我心裡覺得難受啊。放著既正確又簡短的稱呼不用，硬要講一個不倫不類的「小老鼠」，這社會倒底是發生什麼事了？但是不可否認當我屈服於這個文化後，就成了共犯。這讓我極度認知失調，於是開始說服自己「語言這種東西沒有對錯，能有效溝通就是好的」「唸錯唸對不重要，維持溝通效率才重要」

2.每次別人問我「貴姓？」「木易楊嗎？」，我回答前都必定經過一番內心煎熬，然後不得不說「對……木易楊……」但我內心超級難受啊，這分明就是木昜楊啊，為什麼你們這些人都要說是木易楊呢！然而不是我不講「木昜楊」，但是每次得到的反應不外乎是「蛤？」「呃……請問是怎麼寫？」所以最後我還是屈伏了……但木易楊木易楊木易楊一直講，讓我感到極度認知失調，於是我又開始說服自己「語言這種東西沒有對錯，能有效溝通就是好的」「唸錯唸對不重要，維持溝通效率才重要」

積非成是，已是一種時代趨勢。所謂趨勢，就是不管你認不認同，它都是一定會發生的事。
就我而言，不管我「小老鼠」唸得有多溜、「木易楊」講得多順口，內心始終難以接受。
